#  Contact Book

A simple Contact Book application built with **C#** that connects to a **SQL Server database** using **ADO.NET**.

##  Features

- Add new contacts 
- Edit existing contacts 
- Delete contacts 
- Search through contacts 
- Clean and user-friendly interface

##  Technologies Used

- C# (.NET Framework )
- Windows Forms (WinForms)
- SQL Server
- ADO.NET
